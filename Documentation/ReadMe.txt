This extension is a bridge to FLOW3. It requires a (running) FLOW3 installation in your server. I hope this piece of software is usefull for you to get in touch with the great features an concepts of FLOW3.

Maybe you also want to check out the flow3blog extension, which delivers the configuration to access the blog package enclosed in the TYPO3 V5 Distribution.

For more Information, visit http://forge.typo3.org/wiki/extension-thebridge/Manual.
